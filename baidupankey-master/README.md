# 查询百度网盘提取密码

# [Web版](https://pnote.net/pan/)


> 基于云盘万能钥匙API，查询百度网盘分享链接密码。大牛轻喷～

![baidupankey示例截图](https://raw.githubusercontent.com/wu-dada/baidupankey/master/example.png)

---


## 安装

---

```
npm install baidupankey -g
```

## 用法

---

```
bdp -u https://pan.baidu.com/s/xxxxx
```

## 选项

---

-  `-u`, `--url`      **要查询的百度网盘链接**
-  `-h`, `--help`     **显示帮助信息**
-  `-v`, -`-version`  **显示版本号**
